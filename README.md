# My Personal Portfolio Website

Welcome to the repository for my personal portfolio website! This site serves as my digital home base, showcasing my software engineering projects, technical skillset, and academic background as a student.

 [Live Website Link](https://DeanM01.github.io) 

---

##  Design & User Experience (UX) Goals
When building this portfolio, I focused heavily on core front-end performance and clean layout design principles:
* **Mobile-First Responsiveness:** Built using CSS Flexbox and Grid to ensure a seamless reading and navigation experience on everything from smartphones to ultrawide monitors.
* **Frictionless Navigation:** Designed as a sleek, single-page layout with a sticky navbar so recruiters can access my projects and resume in under 3 seconds.
* **Performance-Focused:** Built entirely with raw, semantic HTML5 and vanilla CSS3—no heavy frameworks or external UI library overhead, resulting in instant load times.

## Tech Stack & Tools
* **Front-end:** HTML5, CSS3 (using CSS Variables for clean styling architectures)
* **Design/Wireframing:** Figma
* **Hosting/Deployment:** GitHub Pages

---

## Repository Structure
Here is how the project files are organized:
* `index.html` - The core structure and semantic content of the single-page application.
* `style.css` - Custom responsive layout architecture, typography, and styling variables.
* `resume.pdf` - My up-to-date academic and professional resume.
